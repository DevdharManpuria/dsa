<h3><a href="https://codeforces.com/contest/2189/problem/D1" target="_blank" rel="noopener noreferrer">Little String (Easy Version)</a></h3>

<div class="header"><div class="title">D1. Little String (Easy Version)</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p> </p><p><span class="tex-font-style-bf">This is the <span class="tex-font-style-it">easy</span> version of the problem. The difference between the versions is that in this version, it is guaranteed that the string $$$s$$$ does not contain the character <span class="tex-font-style-tt">?</span>.</span></p><p>For the string $$$w_1w_2 \ldots w_n$$$, consisting of characters <span class="tex-font-style-tt">0</span> and <span class="tex-font-style-tt">1</span>, we define $$$f(w)$$$ as the number of permutations $$$p_1, p_2, \ldots, p_n$$$ of the array $$$[0, 1, \ldots, n-1]$$$, such that for all $$$i$$$ from $$$1$$$ to $$$n$$$ the following holds:</p><ul> <li> if $$$w_i = \texttt{1}$$$, then there exist such $$$1 \leq l \leq r \leq n$$$ that $$$\operatorname{mex}([p_l, p_{l+1}, \ldots, p_r]) = i$$$;$$$^{\text{∗}}$$$<p> </p></li><li> if $$$w_i = \texttt{0}$$$, then there do not exist such $$$1 \leq l \leq r \leq n$$$ that $$$\operatorname{mex}([p_l, p_{l+1}, \ldots, p_r]) = i$$$. </li></ul><p>Given a string $$$s_1s_2 \ldots s_n$$$, consisting of characters <span class="tex-font-style-tt">0</span> and <span class="tex-font-style-tt">1</span>, and a positive integer $$$c$$$. <span class="tex-font-style-bf">Note that in this version of the problem, the string $$$s$$$ doesn't contain <span class="tex-font-style-tt">?</span></span>. Consider all strings $$$w$$$ that can be obtained from $$$s$$$ by replacing all characters <span class="tex-font-style-tt">?</span> with characters <span class="tex-font-style-tt">0</span> and <span class="tex-font-style-tt">1</span>. Find the smallest value of $$$f(w)$$$ among all such strings $$$w$$$ that is not divisible by $$$c$$$, or determine that such a string $$$w$$$ does not exist. Since the answer may be large, find it modulo $$$10^9+7$$$.</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$The minimum excluded (MEX) of a collection of integers $$$c_1, c_2, \ldots, c_k$$$ is defined as the smallest non-negative integer $$$x$$$ which does not occur in the collection $$$c$$$. </p></div></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows. </p><p>The first line of each test case contains two integers $$$n$$$ and $$$c$$$ ($$$3 \leq n \leq 2 \cdot 10^5$$$, $$$1 \leq c \leq 10^9$$$) — the length of the string and the number that limits the value of the function.</p><p>The second line of each test case contains a string of length $$$n$$$, consisting of characters <span class="tex-font-style-tt">0</span> and <span class="tex-font-style-tt">1</span> — the string $$$s$$$.</p><p>It is guaranteed that the sum of $$$n$$$ across all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, if there exists such a string $$$w$$$ that can be obtained from $$$s$$$ by replacing all characters <span class="tex-font-style-tt">?</span> with characters <span class="tex-font-style-tt">0</span> and <span class="tex-font-style-tt">1</span>, such that $$$f(w)$$$ is not divisible by $$$c$$$, then output the minimum value of $$$f(w)$$$ among all such strings $$$w$$$. The answer should be output modulo $$$10^9+7$$$. If such a string does not exist, output $$$-1$$$.</p></div><div class="sample-tests"><div class="section-title">Example</div><div class="sample-test"><div class="input"><div class="title">Input<div
